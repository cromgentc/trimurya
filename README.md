# Trimurya Corporation

Production-oriented enterprise website and REST API for Trimurya Corporation.

## Stack

- Frontend: React, Vite, Tailwind CSS, React Router, Axios, Framer Motion, React Icons, Swiper, AOS, React Hook Form
- Backend: Node.js, Express, MongoDB, Mongoose, JWT, bcrypt, Multer, Nodemailer, Helmet, Morgan, Compression, CORS, Express Validator

## Quick Start

```bash
npm run install:all
npm run dev
```

Frontend runs on `http://localhost:5173`.
Backend runs on `http://localhost:5000`.

Copy `backend/.env.example` to `backend/.env` and update values for production.
